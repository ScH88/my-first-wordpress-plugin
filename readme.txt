Contributors: ScH88
Tags: city, cities
Tested up to: 1.0.0
Stable tag: city


A wordpress plugin I have built myself.

== Description ==

This plugin displays information about a different city every day. Users can also add, delete and edit cities in the plugin's CSV file.

== Installation ==

Copy and Paste this plugin into your website's wp-content/plugins folder, then go to the Wordpress admin interface > plugins to find and activate it.
